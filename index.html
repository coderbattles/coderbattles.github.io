Description

Create a web application using PHP and MYSQL (without any frameworks). This web application will let users register, login, make posts and (un)follow other users.

When a user "follows" another user, they see the posts of the user they follow in addition to the ones they themselves create.

You are expected to have a Registration Page, Login Page, Users Page and Posts Page. The USERS page will provide an interface for users to see all other users registered on the application (except themselves). From that interface, they can "follow" and "unfollow" other users. The POSTS page will provide the interface that lets any logged in user view "posts" of other users they follow, including their posts. These posts will be ordered based on the time they were created, from newer (top) to older (bottom).
You are expected to write tests, collect coverage (Travis) and put a coverage badge in your README and you are to host your project on Heroku and provide the link in your repo. Make sure to write good commits and push to your repo regularily.

After submission, any changes made to your code will be disregarded.
